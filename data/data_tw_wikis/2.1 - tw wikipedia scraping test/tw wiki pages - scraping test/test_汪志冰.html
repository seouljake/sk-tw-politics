汪志冰（1957年4月20日－），台灣女性政治人物，中國國民黨籍，畢業於文藻外語學校德文科、東吳大學德文系畢業、美國東密西根大學文學碩士 ，曾任中華民國全國總工會秘書長。

曾擔任第3屆新黨國大代表，後以親民黨籍及國民黨籍擔任第9、10、11（遞補）、12、13屆臺北市第一選舉區議員（士林區、北投區）[1]，2010年曾因初選落敗退黨競選連任失利[2]，2012年因時任議員陳碧峰解職而重返議會並再度連任至今[3]。2020年參選立委，在國新兩黨協調初選中以些微差距擊敗潘懷宗代表泛藍出征[4]，但最終敗給尋求連任的民進黨立委吳思瑤[5][6][7]。

質詢監督[编辑]

教育文化[编辑]

  1. 針對自動販賣機進入校園，要求教育局加強與學校、家長溝通。
  2. 要求教育局針對108課綱增加家長說明會場次，並盡快擬定108課綱教師評鑑計畫。
  3. 揭露北市文化局接受文化部文資局補助，補助總額僅占6都1%。
  4. 揭露北市9成校園周邊曾發生過毒品案件。
  5. 要求教育局加強性教育，避免學生未預期懷孕，同時保障小媽媽受教權。

交通部門[编辑]

  1. 要求交工處加快北市自動化交通號誌架設速度。
  2. 揭露年輕機車騎士死傷佔整體事故3成。
  3. 要求交工處普查北市路緣斜坡未與行穿道對齊之路口，並分年度進行列管、改善，以落實人本交通。
  4. 揭露車位即時資訊系統狀況百出。

警政衛生[编辑]

  1. 要求市府重視紙尿布垃圾問題。
  2. 要求衛生局加強查緝電子煙，以守護市民健康。
  3. 要求衛生局落實北市食安自治條例14條規定，加強對外燴餐飲業者掌握。
  4. 要求警察局改善北市各警分局女警備勤室安全設備。

工務部門[编辑]

  1. 揭露道管中心成立後，違規案件數量反而上升7倍。
  2. 揭露104-108年防火巷案件數共查報1156件，109年下半年仍有331件未拆除。
  3. 揭露台北流行音樂中心、廣慈公宅等大型公共建設，防火門之燒測實驗室為爭議不斷的「明道大學防火實驗室」。
  4. 揭露聯開及公宅管理權責問題多。

民政部門[编辑]

  1. 要求殯葬處改善二殯周邊交通。
  2. 揭露「網路公祭」使用率低，並要求殯葬處改善。
  3. 要求社會局增加家防中心社福人員工作誘因。
  4. 要求職能發展學院增加青年參訓率、提高招生數並改善課程配套措施。

財政建設[编辑]

  1. 要求動保處增加街貓友善照護行動方案（TCCP）之成效。
  2. 要求校園周邊一定範圍內不得設置娃娃機。
  3. 揭露台電計畫性停電時間過長、通知時間倉促，造成停電之店家營業損失。
  4. 爭取北投設立狗運動公園及狗活動區。
  5. 要求北市制定自助洗衣店管理之自治條例。
  6. 揭露北市府開發APP數量不僅位列六都後段班，柯市府開發數量也不及郝市府時期。
  7. 要求北市盡快制訂動物主題餐廳之自治條例。
